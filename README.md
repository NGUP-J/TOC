## Requrement 

This Web Application Require Bootstrap 5.3.2 to run properly 

## Setting up

1. Clone this git 
    ```git clone https://github.com/NGUP-J/TOC```
2. Install Bootstrap 
    ```npm install bootstrap@v5.3.2```
    OR
    ```yarn add bootstrap@v5.3.2```
3. Use Live server on VSCode to launch
